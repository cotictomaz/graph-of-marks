# Pinned dependency list used to build the Docker images (build/Dockerfile,

# `docker build`) to resolve cleanly with no conflicts, so this file is
# reproducible across cluster nodes/builds, per the "SLURM Web Guide"
# recommendation to pin exact versions. Do not loosen a pin without
# re-verifying the full set resolves (see README_SLURM.md's troubleshooting
# note on ResolutionImpossible errors if you do change something here).
#
# torch/torchvision/torchaudio are pinned to the exact versions vllm==0.8.5

# deliberately an older, more driver-compatible combo than whatever `>=`
# floors would otherwise resolve to unconstrained (which drifted to
# torch 2.11/cu13 wheels needing a very recent NVIDIA driver).

# (RTX 3090 / Titan Xp) image. build/Dockerfile.rtx5090 strips these three
# lines out and installs matching cu128 wheels separately (see that file).
